O DnD Challenge Creator é um software grátis e livre desenvolvido para mestres que tenham pressa 
na construção de monstros e/ou PdM's para suas aventuras em Dungeons and Dragons 3.5.

Atualmente o software encontra-se inutilizável e muito longe até mesmo de uma versão beta para teste. 

Expectativas para um futuro próximo: um database sólido e completo de monstros e talentos dos livros 
bases de D&D; criação semi automática ou automática de fichas para PdM's que sejam satisfatórias com 
as classes pretendidas pelo mestre;

Expectativas para um futuro distante: Dar suporte a criacao de grupos de PdM's juntamente com suas 
estatisticas de combate (pontos fortes e fracos da equipe, melhor situacao de combate, etc).
